Модуль предназначен для CMS ShopOS 2.5.9

Версия модуля: 1.2


Установка модуля:
1) Скопируйте папку ik в папку http://адрес вашего сайта/modules/payment/
2) В разделе панели администратора перейдите в раздел Модули -> Модули оплаты, нажмите на Интеркасса 2.0
3) Все необходимый настройки можно взять с вашей кассы на сайте Интеркассы
	-Укажите оплаченный статус заказа: (здесь укажите какой статус будут иметь заказы поле успешной оплаты с помощью Интеркассы)
4) Далее приведены настройки для самой Интеркассы. В разделе интерфейс добавьте следующие URL

URL Успешной оплаты:http://yourShopOS.com/checkout_process.php
URL Неуспешной оплаты:http://yourShopOS.com/checkout_payment.php
URL Ожидания оплаты:http://yourShopOS.com/checkout_payment.php
URL Взаимодействия: http://yourShopOS.com/modules/payment/ik/ik_process.php

5)На этой же странице запретите переопределение запросов по каждым полем
*Также укажите в настройках вашей кассы в разделе Безопасность проверять ли цифровую подпись, это существенно поможет улучшить безопасность

Желательно:
 - Валюта, в которой магазин передает сумму плетежа на платежный шлюз "Интеркасса".Необходимо указывать КОД валюты. Например USD (доллар)
 - Необходимо выставить доступ к папке и вложеным файлам модуля интеркассы http://yourShopOS.com/modules/payment/ik/ доступ 777(разрешить все)
